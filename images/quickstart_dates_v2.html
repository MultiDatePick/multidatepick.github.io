How it works

Drop this component onto any record page. It reads the record ID automatically. When a user picks dates and clicks Save, the component creates one child record per date in whatever object you specify — setting your chosen date field to the selected date and your chosen lookup field to the current record ID. Turn on "Load Existing Dates" to show previously saved dates as orange dots on the calendar.
