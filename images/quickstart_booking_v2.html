How it works

This component uses two objects: a Resource object (any object where each record is something bookable — rooms, equipment, vehicles, etc.) and a Booking object (where reservations are stored). Drop it on any record page, wire up the field mappings, and the component populates the resource dropdown from your resource records. When a user selects a resource, picks dates, and clicks time slots, one booking record is created per slot with all fields filled in. Existing bookings automatically show as unavailable on the time grid.
