Common Flow Pattern

Screen → Parse → Loop → Create Records

  1. Add a Screen with the MultiDatePick component. Enable JSON output.
  2. Add an Action: "Parse Date Entries from JSON" — pass the JSON variable.
  3. Add a Loop over parsedCollection.data.
  4. Inside the loop, create records using fromDate, toDate, startTime, endTime.

MultiDatePickEntry Fields

FieldTypeDescription
idIntegerSequential identifier (1, 2, 3...)
fromDateDateStart date of the range
toDateDateEnd date (same as fromDate for single dates)

endTimeTimeEnd time (DateTime and Booking only)
